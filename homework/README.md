Задание 1: Создайте кастомный React-хук для управления формой

Ваш задача - создать кастомный хук useInput, который позволит обрабатывать и хранить значение внутри инпута. Хук должен принимать начальное значение инпута и возвращать объект, содержащий текущее значение инпута и функцию для обновления этого значения.

Кроме того, хук должен следить за состоянием ввода («dirty», «visited»), чтобы показывать валидацию пользовательского ввода только когда необходимо.

Задание 2: Создайте кастомный хук для работы с API

Создайте кастомный хук useFetch, который примет URL в качестве параметра, выполнит запрос к этому URL и вернет данные, полученные от сервера, а также состояния загрузки и ошибки, если они произошли.

Задание 3: Измените существующий проект с использованием кастомных хуков

Выберите одну из ваших предыдущих работ на React и найдите часть кода, которую можно повторно использовать и упростить с помощью кастомного хука. Создайте этот кастомный хук и примените его на практике, улучшив код вашего проекта.

Задание 4: Создайте кастомный хук для работы с хранилищем localStorage

Создайте кастомный хук useLocalStorage, который поможет вам работать с хранилищем браузера localStorage - записывать в него данные, читать данные оттуда и удалять данные. Ваш хук должен корректно обрабатывать все возможные исключения, которые могут возникнуть при работе с localStorage.
